#  Foretag ESLint Config

The official eslint configuration for Foretag internal and public projects.

## Install

**NPM**

```npm install @foretag/eslint-config --save-dev```

**Yarn**

```yarn add -D @foretag/eslint-config```

## Usage
.eslintrc
```json
{
	"extends": "@foretag"
}
```

## Supported languages and tools

- [x] JavaScript
- [x] TypeScript
- [x] GraphQL
- [x] React
- [x] Cypress
- [x] Jest
